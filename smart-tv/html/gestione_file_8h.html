Metodo di lettura di un record da un file.

Questa funzione serve a generalizzare l'operazione di lettura di un record dall'interno di un file Oltre al riferimento del file (*file) e al riferimento dell'oggetto (*data), vengono richiesti in input alcuni dati necessari per il corretto posizionamento all'interno del file

  • l'identificativo univoco intero dell'oggetto da scrivere (*id)
  • la dimensione della struttura dati (dimensione_struct)
Parametri
*filePuntatore del file
*idIdentificativo univoco intero dell'oggetto
dimensione_structDimensione della struttura dati
*dataPuntatore all'oggetto da scrivere all'interno del file
